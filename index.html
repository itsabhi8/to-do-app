To-do List

    Key Features of Our Todo List App

    • Quick Task Addition: Easily add new tasks using the input field and “Add” button, or by pressing Enter.
    • Instant Task Deletion: Remove tasks effortlessly with a delete button next to each item.
    • Organized Task Display: View tasks in a clean, structured list format for easy management.
    • Responsive Design: Enjoy a modern, adaptable interface that works seamlessly on various devices.
    • User-Friendly Interface: Simple controls and visual feedback enhance ease of use and navigation.
    • Task Prioritization: Set task priority levels to categorize tasks by importance.
    • Due Dates & Times: Assign due dates and times to tasks and view them accordingly.
    • Reminders: Set reminders for tasks to get notified before the due time.
    • Task Completion: Mark tasks as done with a checkbox and visually differentiate completed tasks.
    • Task Status Filtering: View tasks filtered by their status: completed or cancelled.
